# RedDuck Academy — Engineering Principles Template

Template repo for the Engineering Principles course by [RedDuck](https://redduck.io). The
lessons and submission live on [academy.redduck.io](https://academy.redduck.io). Clone this
repo, work through the tasks in the language of your choice, push to your own fork, and submit
the repository URL on the academy website.

## How it works

Each task is a folder with a `TASK.md` brief and a `SOLUTION.md` you fill in. Coding tasks also
have a `src/` folder with a starter file in each supported language.

Pick one language and use it across the tasks: **TypeScript**, **Python**, **Java**, or **C#**.
Complete the starter for your language and leave the other languages as they are. Your work is
reviewed by reading the files you push, so nothing needs to build or run.

## Workflow per task

1. Read the task's `TASK.md`.
2. For a coding task, complete the starter file for your language in `src/`.
3. Write your answer in `SOLUTION.md`.
4. Commit, push to your fork, and submit the repository URL on
   [academy.redduck.io](https://academy.redduck.io).
